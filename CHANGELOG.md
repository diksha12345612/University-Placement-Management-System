# 📓 Project Changelog

All notable changes to the **University Placement & Preparation Portal** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-03-30
### Added
- Created `CHANGELOG.md` and enhanced `CONTRIBUTING.md` with professional standards.
- Implemented `useNotifications` custom hook for centralized state management.
- Added root `package.json` for unified workspace management.
- Integrated `LoadingSpinner` global state for improved UX.

### Changed
- **[Refactor]** 30-commit modernization cycle completed for the entire repository.
- Centralized server-side middleware (security, cors, rate limiting, error handling).
- Standardized API route naming conventions (`resume.js`, `admin.js`).
- Synchronized Tailwind CSS tokens with global CSS variables.
- Improved accessibility (A11Y) across all major UI components.

### Fixed
- Resolved React dependency warnings in `AuthContext`.
- Standardized cross-platform line endings and file formats.
- Hardened server startup with environment variable validation.
